### 📍 页面类型说明

| 类型 | 页面 | 特点 |
|------|------|------|
| **主页面** | Home / Summary | 核心入口，包含 Daily Meal / Exercise / Scanned 三个 Tab |
| **主页面** | Home / Health Score | 从首页**左右滑动**进入，健康数据展示 |
| **功能页** | Scan Menu | 底部导航 Scan 按钮进入，支持三种扫描模式 |
| **引导层** | First Time Guide | 首次使用时覆盖在 Scan 上方 |
| **教程页** | Help Tutorial | 从 Scan 页帮助按钮进入，3页独立教程 |

---

### 🔄 关键用户路径

#### 路径 1: 新用户首次扫描
```
首页 → 点击 Scan → First Time Guide (覆盖层) → Scan now → 扫描
```

#### 路径 2: 老用户日常扫描
```
首页 → 点击 Scan → 直接进入扫描界面 → 拍照/选图
```

#### 路径 3: 查看帮助
```
Scan Menu → 点击右上角 ? → Help Tutorial P1 → P2 → P3 → Scan Now → 返回扫描
```

#### 路径 4: 查看健康评分
```
首页 (Summary) → 左右滑动 → Health Score 页
```

#### 路径 5: 食物录入流程
```
Scanned (草稿箱) → 确认食物 → 移动到 Daily Meal → 数值更新到卡路里面板
```

#### 路径 6: 查看/编辑食物详情
```
方式 A: Scan Menu → 点击右下角缩略图 → Food Details 页
方式 B: 首页 Scanned Tab → 点击某个食物卡片 → Food Details 页
```

#### 路径 7: 记录到指定日期
```
Food Details → 点击 "Log to date" → 日期选择器 → 选择日期 → Done → 记录完成
```

### 📊 Tab 功能定义

| Tab | 数据性质 | 作用 |
|-----|----------|------|
| **Daily Meal** | 已确认 | 当天已确认的食物记录，影响卡路里计算 |
| **Exercise** | 已确认 | 当天已确认的运动记录，影响卡路里计算 |
| **Scanned** | 草稿 | 扫描队列/草稿箱，需要用户确认后才进入 Daily Meal |
