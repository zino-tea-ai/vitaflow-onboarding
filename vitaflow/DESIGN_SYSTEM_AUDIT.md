### 🔄 关键用户路径

#### 路径 1: 新用户首次扫描
```
Home → 点击 Scan → First Time Guide (覆盖层) → Scan now → 扫描
```

#### 路径 2: 老用户日常扫描
```
Home → 点击 Scan → 直接进入扫描界面 → 拍照/选图
```

#### 路径 3: 查看帮助教程
```
Scan Menu → 点击 ? → P5: 扫描技巧 → P6: AI分析 → P7: 准确性 → 返回
```

#### 路径 4: 查看健康评分
```
Home / Summary → 左右滑动 → Home / Health Score
```

#### 路径 5: 食物录入流程
```
Scanned Tab → 确认食物 → 移动到 Daily Meal → 更新卡路里面板
```

#### 路径 6: 查看/编辑食物详情
```
Scan Menu (缩略图) → Food Details → [Log to Date / Change Name / Edit Calories]
```

#### 路径 7: 记录运动
```
Home (Exercise Tab) → 选择运动类型 → 设置时长/日期/时间 → Save Exercise
```

#### 路径 8: 查看进度
```
底部导航 Progress → 查看体重/营养素变化趋势
```

#### 路径 9: 个人设置
```
底部导航 Profile → [Personal Details / Preferences / Language / Nutrition Goals / Weight History]
```

#### 路径 10: 退出/删除账户
```
Profile → Logout → 确认退出
Profile → Delete Account → 确认删除
```

### 📊 Tab 功能定义

| Tab | 数据性质 | 作用 |
|-----|----------|------|
| **Daily Meal** | 已确认 | 当天已确认的食物记录，影响卡路里计算 |
| **Exercise** | 已确认 | 当天已确认的运动记录，影响卡路里计算 |
| **Scanned** | 草稿 | 扫描队列/草稿箱，需要用户确认后才进入 Daily Meal |
