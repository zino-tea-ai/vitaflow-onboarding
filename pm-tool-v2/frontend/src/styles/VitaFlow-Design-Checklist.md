## 🔍 快速检查技巧

### Figma 快捷键
| 操作 | 快捷键 |
|------|--------|
| 查看间距 | 按住 `Alt` 移动鼠标 |

| 查看 HSB | 颜色选择器 → 切换到 HSB |

```
X, Y, Blur, Spread
例如：0, 4, 12, 0 = X偏移0, Y偏移4, 模糊12, 扩展0
```
