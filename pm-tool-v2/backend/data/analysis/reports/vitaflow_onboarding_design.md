# VitaFlow Onboarding 设计建议

## 🎯 设计目标

基于对 15 个竞品 App 的分析，为 VitaFlow 设计一个**高转化率**的 Onboarding 流程。

### 核心指标目标

| 指标 | 行业平均 | 目标值 |
|------|----------|--------|
| Onboarding 完成率 | 40-60% | 70%+ |
| Install → Trial | 15-25% | 35%+ |
| Trial → Paid | 20-30% | 40%+ |

---

## 📋 推荐流程设计

### 总体规划

- **总步骤数**：50 步（±10）
- **预计完成时间**：5-8 分钟
- **付费墙位置**：85% (第 42-45 步)

## 🎨 视觉设计规范

### 配色建议

| 用途 | 颜色 | 说明 |
|------|------|------|
| 主色 | #4CAF50 | 绿色，代表健康 |
| 强调色 | #FF9800 | 橙色，用于 CTA |
| 背景 | #FFFFFF | 白色，简洁 |
| 文字 | #212121 | 深灰，易读 |
| 次要文字 | #757575 | 灰色 |

### 字体建议

- **标题**：SF Pro Display Bold, 24px
- **正文**：SF Pro Text Regular, 16px
- **按钮**：SF Pro Text Semibold, 17px

### 动效建议

- **页面切换**：slide from right, 300ms
- **按钮点击**：scale 0.95, 100ms
- **进度条**：ease-out, 500ms
