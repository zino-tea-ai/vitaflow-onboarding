## 🎨 视觉风格

### 整体风格
选择一个主要风格：
- [ ] 🍎 **极简现代** - 大量留白，简洁线条（Apple 风格）
- [ ] 🌈 **活泼多彩** - 鲜艳色彩，圆角卡片（Duolingo 风格）
- [ ] 📊 **专业商务** - 清晰层级，数据驱动（Notion 风格）
- [ ] 🎮 **暗黑科技** - 深色背景，霓虹强调（游戏/极客风格）
- [ ] 🌿 **自然健康** - 柔和色彩，自然元素（Calm 风格）
- [ ] 🏛️ **优雅奢华** - 金色点缀，精致细节（高端品牌）
- [ ] 📰 **内容优先** - 阅读友好，信息密度高（Medium 风格）
